
# Nexus AI Assistant  

Nexus AI is a sleek, modern web application designed to act as a versatile AI assistant. Built with a robust Python/Flask backend and a beautifully styled, responsive frontend, it offers a seamless interface for interacting with multi-LLM-inspired capabilities.  

---

## Features  

* **Multi-Mode Intelligence:** Switch effortlessly between three distinct operational modes:  
* **Q&A Mode:** Get clear, accurate answers from deep factual knowledge bases.  
* **Summarize Mode:** Condense long texts into crisp, structured summaries.  
* **Creative Mode:** Generate unique stories, essays, or creative content from simple prompts.  


* **Modern UI/UX:** A premium dark-themed interface featuring a glassmorphism header, gradient accents, and smooth scroll behaviors.  
* **Interactive Chat Interface:** Real-time chat simulation with loading states, user/AI message distinction, and auto-scrolling.  
* **Feedback Loop:** Built-in mechanisms for users to rate AI responses, logging feedback directly to the backend for future model improvements.  

---

## Tech Stack  

* **Backend:** Python, Flask, Gunicorn (Production server)  
* **Frontend:** HTML5, CSS3, Vanilla JavaScript  
* **Environment Management:** `python-dotenv`  
* **Icons:** FontAwesome

## Installation & Setup  

Follow these steps to run the Nexus AI Assistant locally on your machine.  

**1. Clone the repository**  

```bash
git clone https://github.com/yourusername/nexus-ai-assistant.git  
cd nexus-ai-assistant  

```

**2. Create a virtual environment**  

```bash
python -m venv venv  
source venv/bin/activate  # On Windows use: venv\Scripts\activate  

```

**3. Install dependencies**  

```bash
pip install -r requirements.txt  

```

**4. Environment Variables**  
Create a `.env` file in the root directory and add any necessary API keys (e.g., OpenAI API key if you uncomment the production API calls in `app.py`).  

```env
# .env file   
API_KEY=your_api_key_here  

```

**5. Run the application**  

```bash
python app.py  

```

The application will be available at https://nexus-ai-app-dnle.onrender.com/

## Usage  

1. Open the application in your web browser.  https://nexus-ai-app-dnle.onrender.com/  
2. Select your desired **Model Mode** (Q&A, Summarize, or Creative) from the dropdown in the chat control panel.  
3. Type your prompt into the input area and hit **Send** (or press Enter).  
4. Provide feedback on the generated response by clicking the **Helpful?** button attached to the AI's reply.
